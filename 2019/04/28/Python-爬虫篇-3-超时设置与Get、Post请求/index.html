超时设置

当我们使用爬取网页时，我们可以设置超时时间，以便可以更快或者重新访问网页，从而避免在获取响应网页内容上浪费太多时间

语法如下：

urllib.request.urlopen(timeout=1) # 设置1秒为超时界限
